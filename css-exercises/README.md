# CSS Practice Exercises - Complete Collection

Prepared by: **SHERILENE B. PAMINTUAN, MIT**  
Organized and Enhanced by: CSS Practice Exercises Project

## 📚 Overview

This comprehensive collection contains **26 CSS practice exercises** covering everything from basic styling to advanced responsive design and animations. Each exercise is in a **separate file** (not monolithic) for better organization and reusability.

## 🎯 Exercise Categories

### Main Exercises (01-12)

1. **01-profile-page.html** - Simple personal profile with centered content, profile picture, and hobbies list
2. **02-navigation-menu.html** - Horizontal navigation bar with hover effects and inline styling
3. **03-product-card.html** - Product card with border, shadow, and styled button with hover state
4. **04-responsive-landing.html** - Landing page with header, hero section, features, and footer using Flexbox
5. **05-login-form.html** - Centered login form with email/password inputs and rounded styling
6. **06-image-gallery.html** - CSS Grid gallery with 6 images and hover zoom effect
7. **07-portfolio-website.html** + **07-portfolio-website.css** - Multi-section portfolio with smooth scrolling
8. **08-dashboard-ui.html** + **08-dashboard-ui.css** - Dashboard layout with sidebar and responsive grid
9. **09-netflix-clone.html** + **09-netflix-clone.css** - Netflix-style landing page with hero section
10. **10-responsive-website.html** + **10-responsive-website.css** + **10-responsive-website.js** - Fully responsive with hamburger menu
11. **11-dark-light-mode.html** + **11-dark-light-mode.css** + **11-dark-light-mode.js** - Theme toggle with CSS variables
12. **12-animated-webpage.html** + **12-animated-webpage.css** - CSS animations with fade-in effects

### Pamintuan's CSS Fundamentals (13-26)

13. **13-css-background-color.html** - Background color styling examples
14. **14-css-background-opacity.html** - Opacity and transparency using RGBA
15. **15-css-background-image.html** - Background images with repeat-x, repeat-y, no-repeat
16. **16-css-borders.html** - Different border styles (solid, dashed, dotted, double, grooved)
17. **17-css-rounded-borders.html** - Border radius and circular elements
18. **18-css-margins.html** - Margin properties and auto-centering
19. **19-css-paddings.html** - Padding variations and sizing
20. **20-css-box-model.html** - Complete box model explanation with examples
21. **21-css-links.html** - Link states (:link, :visited, :hover, :active)
22. **22-css-tables.html** - Basic table styling with borders and hover effects
23. **23-css-responsive-tables.html** - Responsive tables with media queries
24. **24-css-positioning.html** + **24-css-positioning.css** - Absolute positioning and z-index
25. **25-css-classes.html** - CSS classes and multiple class usage
26. **26-css-table-gallery.html** - Image gallery using HTML tables and CSS

## 🚀 How to Use

### For Learning:
- Start with exercises 01-06 for basics
- Move to 07-12 for intermediate concepts
- Review 13-26 for fundamental CSS properties

### For Reference:
- Each file is **self-contained** and can be opened independently in a browser
- Inline CSS or linked CSS files are provided
- All examples use placeholder images (via.placeholder.com) for quick testing

### For Development:
- Copy any exercise as a starting point
- Modify and experiment with the CSS
- Use as templates for your own projects
